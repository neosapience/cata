# cata
`Ca`llback `t`esting `a`pp

> [iamport](https://www.iamport.kr/) 같은 결제모듈에서 callback api를 테스트하기 위한 웹서비스


## API endpoints for admin

* `GET` /api/cb
* `POST` /api/cb
* `GET` /api/cb/{id}
* `DELETE` /api/cb/{id}
* `GET` /api/cbit

## API endpoints for callback posting

* `POST` /api/cb/{id}
